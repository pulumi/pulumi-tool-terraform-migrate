## Description

This is a comprehensive AWS infrastructure example demonstrating a full DNS-to-Database architecture including:
- VPC with public/private subnets
- Application Load Balancer (ALB)
- EC2 instances (bastion + private app servers)
- RDS MySQL database
- Route53 DNS registration
- ACM certificate management
- Security groups for each tier
- Elastic IPs
